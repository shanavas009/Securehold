# SecureHold (Landlords) static export

This folder is ready to upload to a GitHub repo and use with GitHub Pages.

## Files
- `landlords.html` – the Landlords page
- `landlords_files/` – all images/scripts saved by the browser
- `favicon.png` – created from the logo (optional)

## GitHub Pages
1. Upload everything in this folder to your repo root
2. In GitHub: Settings → Pages → Build and deployment → Source: **Deploy from a branch**
3. Branch: `main` and folder: `/ (root)`

Then browse:
- `.../landlords.html`

Note: Because this was saved from a builder preview, any interactive React behaviour (menus/forms) may not work unless you export the full build (Vite/React) and deploy it. This version is intended as a static HTML snapshot.
